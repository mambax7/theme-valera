Valera Bootstrap Theme For XOOPS CMS 2.5.8.x
=====

![Theme Preview](/theme_preview.png)

Valera is an awesome bootstrap one page site template equipped with lot of modern features such as video background, minimalist slider, parallax and css3 animations. 

Ported to XOOPS [CMS 2.5.8.x](http://xoops.org/modules/news/article.php?storyid=6762)
